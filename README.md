# Agrinho 2026 - Agro e Tecnologia

Este projeto foi desenvolvido para o Concurso Agrinho 2026 com o tema:
**"Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente"**

## 💡 Objetivo
Demonstrar como a tecnologia pode ajudar a melhorar o agronegócio, aumentando a produtividade e reduzindo impactos ambientais.

## 🌱 Tecnologias no agro
- Sensores no solo
- Drones para monitoramento
- Inteligência artificial
- Irrigação inteligente

## 🖥️ Funcionalidades
- Página informativa sobre o tema
- Quiz interativo sobre tecnologia no agro
- Design responsivo e educativo

## 🚀 Como executar
Basta abrir o arquivo `index.html` no navegador.
